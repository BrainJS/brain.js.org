# Brain.js.org

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Deploys to gh-pages branch of Brain.js.org

```bash
npm run deploy
```

### Lints and fixes files

```bash
npm run lint
```
